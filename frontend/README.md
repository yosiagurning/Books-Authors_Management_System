# Frontend

Dokumentasi utama proyek ada di [README.md](file:///d:/Internship/Books&Authors_Management_System/README.md).

Folder ini berisi aplikasi Vue.js untuk Book & Author Management System dengan stack:

- Vue 3
- Vue Router
- Vite
- Tailwind CSS v4
- TypeScript

Command yang paling sering dipakai:

```bash
npm install
npm run build
node ./node_modules/vite/bin/vite.js --host 127.0.0.1 --port 5173
```

Catatan:

- Binary dijalankan langsung lewat `node` agar lebih aman pada path Windows yang mengandung karakter `&`.
